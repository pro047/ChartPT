# ChrtPT
물리치료사들을 위한 환자 평가 관리 시스템입니다

## 개요
환자의 통증 부위, 가동 범위, 움직임 데이터를 기반으로 평가
추후 환자 치료 계획, 운동 계획 등 관리

## 기술 스택
- Frontend : Next.JS, React
- Backend : Node.JS(Express)
- Databas : MySQL
